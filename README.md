## Featured Projects

### RenewedDubai

Production e-commerce platform for refurbished electronics in the UAE.

`Next.js` `NestJS` `TypeScript` `PostgreSQL` `Prisma` `Redis`

- Product, variant and inventory management
- Checkout, orders and admin workflows
- AI-assisted SEO generation
- Google Merchant and structured-data support

---

### PriceScout AI

UAE electronics price-comparison platform using multiple marketplace providers.

`Next.js` `NestJS` `Playwright` `Redis` `PostgreSQL`

- Multi-provider marketplace search
- Product parsing and offer normalization
- Strong product-matching logic
- Provider timeout and failure isolation

---

### AI Camera Detection System

Android camera-monitoring system for driver-safety detection.

`Kotlin` `Android` `RKNN` `RTSP` `MediaCodec`

- Fatigue and mobile-usage detection
- Seatbelt and distracted-driving detection
- Event threshold, debounce and cooldown controls
- JPEG and video evidence generation
